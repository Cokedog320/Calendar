# Pager 关键可观测点

适用范围：月历已切换为 `HorizontalPager` 后的真机回归。

## 通过标准

- 左右滑月时，只应平滑切换到相邻月份，标题月份和网格月份始终一致。
- 点击顶部左右箭头时，每次只移动一个月，不抖动、不回弹、不连跳。
- 点击“今天”后，应回到当前月并高亮今天；若打开了日详情，应一并回到正常首页状态。
- 通过日期跳转后，应落到目标月并选中目标日期，标题和网格同步更新。
- 点击跨月补位日期时，应进入该日期所属月份，选中态落在被点的那一天。
- 打开日详情、设置页、备注页时，再去滑月不应误触关闭、误翻页或把底部 sheet 状态弄乱。

## 重点记录

- 标题月份是否和当前页一致。
- 滑动过程是否有闪跳、卡顿、回弹。
- 选中态是否丢失或落到旧月份。
- 底部 sheet 是否在翻页时保持稳定。

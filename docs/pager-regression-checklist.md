# Pager 月历回归清单

适用范围：当前 Jetpack Compose 月历已切换为 `HorizontalPager` 翻页后的交互回归。

## 测试前提

- 使用最新 `debug` 包。
- 至少准备一台真机，优先 Android 12+。
- 进入首页后确认当前月、农历、节假日、排班角标和备注点均正常显示。

## 核心回归项

1. 首页左右滑月
- 连续向左、向右滑动月历，确认每次只切换一个月。
- 期望：翻页手感接近系统日历，页面不抖、不回弹错页，月份标题与实际页一致。

2. 顶部箭头切月
- 分别点击“上一个月”和“下一个月”多次。
- 期望：每次只移动一个月，标题、月历网格和当前月份状态同步更新。

3. 今天按钮
- 在任意月份下点击“今天”。
- 期望：立刻回到当月，并高亮今天；若当时打开了日详情，能一起回到正常首页状态。

4. 日期跳转
- 通过顶部日期跳转选择一个远月日期。
- 期望：页面跳到目标月，选中目标日期，月标题更新正确。

5. 跨月补位日期点击
- 点击当前月首尾两侧的补位日期格子。
- 期望：能正常进入该日期所在月份，且选中态落在被点击的日期上。

6. 日详情与翻页共存
- 点开某一天的日详情后，尝试左右滑月和点顶部箭头。
- 期望：底部弹层行为正常，不出现误翻页、误关闭或选中态错乱。

7. 设置页与翻页共存
- 打开设置页，再尝试左右滑月与点顶部按钮。
- 期望：设置页优先保持可预期，不出现底层月历状态跳变。

8. 备注页与翻页共存
- 打开备注中心，执行搜索、按月份筛选、点选备注日期跳转。
- 期望：跳转后能落到对应月份，备注筛选结果与跳转目标一致。

9. 连续操作稳定性
- 依次执行“滑月 -> 跳转日期 -> 今天 -> 打开备注 -> 返回首页 -> 再滑月”。
- 期望：没有卡死、错页、月份回跳、状态残留。

## 建议记录

- 记录是否出现标题月份与页面不一致。
- 记录是否存在滑动过程中页面闪跳或回弹。
- 记录底部 sheet 打开时手势是否容易误触翻月。
- 若发现问题，优先保留复现步骤和目标月份截图。
